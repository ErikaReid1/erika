# Simple App - Erika

This is a simple application to serve as a starting point for projects.

## Structure
- **index.html**: Main landing page.
- **css/style.css**: Stylesheet for basic design.
- **js/script.js**: JavaScript for functionality.

## How to Use
1. Clone the repository:
   ```
   git clone https://github.com/ErikaReid1/erika.git
   ```
2. Open `index.html` in your browser to view the app.
